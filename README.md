# Level Up Tuts | React Hooks for Everyone

1. `git clone https://github.com/narimanadam/Leveluptuts-React-hooks-for-everyone.git`
2. `cd Leveluptuts-React-hooks-for-everyone`
3. run `npm install`
4. run `npm start`
